## 🎯 Descripción

AlgoVision es un simulador interactivo que permite visualizar el funcionamiento de algoritmos y estructuras de datos en tiempo real. Ideal para estudiantes que quieren entender cómo funcionan los algoritmos paso a paso.

---

## 🚀 Características

| Módulo               | Algoritmos/Estructuras                                                     |
| -------------------- | -------------------------------------------------------------------------- |
| 🔍 **Búsqueda**      | Lineal, Binaria, Interpolación, Exponencial                                |
| 📊 **Ordenamiento**  | Bubble, Selection, Insertion, Shell, Counting, Radix, Bucket, Quick, Merge |
| 🌳 **Árboles**       | ABB, AVL, Rojo-Negro, B, B+, Recorridos (In, Pre, Post, BFS)               |
| 🔗 **Grafos**        | BFS, DFS, Dijkstra, PRIM                                                   |
| 🏆 **Comparaciones** | Compara rendimiento de todos los algoritmos con gráficos y estadísticas    |

## 📦 Instalación local

```bash
# Clonar el repositorio
git clone https://github.com/ELIDAN17/Simulador-algoritmos
cd AlgoVision

# Crear entorno virtual
python -m venv venv
source venv/bin/activate  # Linux/Mac
# o
venv\Scripts\activate     # Windows

# Instalar dependencias
pip install -r requirements.txt

# Ejecutar
streamlit run app.py
```
