# AlgoVision – Simulador de Algoritmos y Estructuras de Datos

**Curso:** Algoritmos y Estructuras de Datos  
**Universidad Nacional del Altiplano**  
**Autoria:** Ely Johan MVP

## 🚀 Características

- 🔍 **Búsquedas:** Lineal, Binaria, Interpolación, Exponencial
- 📊 **Ordenamientos:** Bubble, Selection, Insertion, Shell, Counting, Radix, Bucket, Quick, Merge
- 🌳 **Árboles:** ABB, AVL, Rojo-Negro, B/B+, Recorridos
- 🔗 **Grafos:** BFS, DFS, Dijkstra, PRIM

## 📦 Instalación

```bash
git clone <repo-url>
cd AlgoVision
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
pip install -r requirements.txt
streamlit run app.py
```
